# GMManager
Allows you to change your Gamemode with ease!

# Commands
/gmc - creative mode

/gms - survival mode

/gma - adventure mode

/gmv - spectator mode

# Permissions
/gms eg.gms

/gms eg.gms.other

/gmc eg.gmc

/gmc eg.gmc.other

/gma eg.gma

/gma eg.gma.pther

/gmv eg.gmv

/gmv eg.gmv.other
