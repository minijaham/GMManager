# GMManager
Allows you to change your Gamemode with ease!

# Commands
/gmc - creative mode

/gms - survival mode

/gma - adventure mode

/gmv - spectator mode

# Permissions
/gms gm.gms

/gms gm.gms.other

/gmc gm.gmc

/gmc gm.gmc.other

/gma gm.gma

/gma gm.gma.pther

/gmv gm.gmv

/gmv gm.gmv.other
